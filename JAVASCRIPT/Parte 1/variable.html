Definiendo variables en una sola linea, no recomendado :D.

let nombre = "shirley", apellidos = "Oxa Cacya";

Mejor en diferentes lineas

let nombre1 = "shirley";

let apellidos2 = "Oxa Cacya";

Para nombrar variables solo se puede incluir letras, dígitos, o los símbolos $ y _. El primer carácter no puede ser un dígito.

let $name="Juan";
let var3 = 5;
let _tem = 45;
let var5hi = "hu";

const variable; para crear variables constantes, es decir inmutables.
